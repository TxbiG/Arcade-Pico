# Arcade-Pico
Small Project with a few games to run on the Pico micro-controller

## Features
- Tetris
- Pong
- Snake

- Todo
  - Breakout
  - Pac man

#### Software
- [Pico-ssd1306 by daschr](https://github.com/daschr/pico-ssd1306)
- [pico-sdk](https://github.com/raspberrypi/pico-sdk)

#### Hardware
- ```Screen```: 0.96" OLED Display Module (128x64) ssd1306 (White)
- ```Microcontroller-board```: Raspberry Pi Pico
- ```Buttons```: Tactile Switch Buttons (12mm square, 6mm tall) x 10 pack
- ```Speaker```: Mini Oval speaker (Adafru)
- ```Battery```: Lithium-ion battery small
